# 🌍 Global AI Job Market 2025 – Excel & Power BI Dashboard

## 📊 Project Overview
This project analyzes 15,000+ global AI job postings to uncover trends in:
- Job demand by role
- Salary distribution (Average & Median)
- Experience-level hiring patterns
- Remote vs On-site trends
- Skill demand frequency
- Monthly hiring trends

The dashboard transforms raw job market data into an interactive Business Intelligence solution using Excel and Power BI.

---

## 🚀 Key Metrics

- 📌 Total Job Openings: 15,000+
- 💰 Average Salary: $115K
- 📈 Median Salary: $99K
- 🌎 Multi-country salary benchmarking
- 📊 Experience-level segmentation (Entry, Mid, Senior, Executive)

## 📊 Business Insights Generated

- Python and SQL dominate AI hiring requirements.
- Senior-level roles command significantly higher compensation.
- Remote roles show strong presence in AI hiring trends.
- Salary variations indicate geographic and experience-level impact.
- Mid-year hiring peaks suggest seasonal recruitment surges.
